# 🍕 Pizza Sales Analysis Dashboard

## 📌 Project Overview

The **Pizza Sales Analysis Dashboard** is a complete business intelligence and data analytics project created using Microsoft Excel. The project focuses on analyzing pizza sales data to discover meaningful business insights, sales trends, customer preferences, and operational performance.

This dashboard helps businesses understand:

* Total revenue generated from pizza sales
* Customer ordering behavior
* Best-selling and worst-selling pizzas
* Sales contribution by pizza category and size
* Daily and monthly sales trends
* Key performance indicators (KPIs)


# 📊 Dashboard Preview

The dashboard contains multiple analytical sections including:

* KPI Summary
* Daily Order Trends
* Monthly Sales Trends
* Sales by Pizza Category
* Sales by Pizza Size
* Top Performing Products
* Worst Performing Products
* Revenue Analysis

The dashboard is designed to provide a professional and interactive reporting experience for business decision-making.

# 📈 Key Performance Indicators (KPIs)

The dashboard tracks important KPIs such as:

## ✅ Total Revenue

Displays the total income generated from all pizza sales.

## ✅ Total Orders

Shows the total number of customer orders.

## ✅ Total Pizzas Sold

Represents the total quantity of pizzas sold.

## ✅ Average Order Value

Measures the average amount spent per order.

## ✅ Average Pizzas Per Order

Shows customer purchasing behavior.

These KPIs help businesses evaluate overall sales performance efficiently.
